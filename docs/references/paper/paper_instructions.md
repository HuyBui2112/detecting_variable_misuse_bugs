| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| Bắt buộc       | **Global Relational Models of Source Code** — Hellendoorn et al., ICLR 2020                                                                  | Tương ứng với “Great: Graph-based Representation Attentional Network for Variable Misuse”. Tên đúng của mô hình là **GREAT: Graph Relational Embedding Attention Transformer**, không phải “Graph-based Representation Attentional Network”. | PDF trực tiếp: ([Vincent Hellendoorn][1]) / OpenReview: ([OpenReview][2])                    |
| Bắt buộc       | **Learning to Represent Programs with Graphs** — Allamanis, Brockschmidt, Khademi, ICLR 2018                                                 | Tương ứng tốt nhất với “Variable Misuse Detection with Relational Graph Attention Networks”. Đây là paper nền tảng cho **VarMisuse** bằng graph program representation + GGNN.                                                               | PDF trực tiếp: ([Miltos Allamanis][3]) / OpenReview: ([OpenReview][4]) / arXiv: ([arXiv][5]) |
| Bắt buộc       | **Self-Supervised Bug Detection and Repair** — Allamanis, Jackson-Flux, Brockschmidt, NeurIPS 2021                                           | Tương ứng với “Self-Supervised Learning for Program Analysis with GNNs”. Paper này dùng BugLab để học phát hiện/sửa bug từ dữ liệu tự tạo, rất hợp phần mở rộng của đề tài.                                                                  | PDF trực tiếp: ([NeurIPS Proceedings][6]) / arXiv: ([arXiv][7])                              |
| Nên đọc        | **Graph Neural Networks in Program Analysis** — Allamanis                                                                                    | Phù hợp với phần cơ sở lý thuyết: vì giải thích GNN cho program analysis, có nhắc trực tiếp **variable misuse detection** và **type inference**.                                                                                             | PDF trực tiếp: ([GNNBook][8])                                                                |
| Nên đọc        | **Deep Learning for Source Code Modeling and Generation: Models, Applications and Challenges** — Le, Chen, Babar, ACM Computing Surveys 2020 | Tương ứng với “Deep Learning for Source Code Modeling: A Survey”. Đây là tên chính thức gần nhất và đáng tin cậy nhất.                                                                                                                       | arXiv: ([arXiv][9]) / PDF trực tiếp: ([arXiv][10]) / ACM: ([ACM Digital Library][11])        |
| Nên đọc        | **A Survey of Machine Learning for Big Code and Naturalness** — Allamanis et al., ACM Computing Surveys 2018                                 | Survey nền tảng về machine learning cho source code, nên dùng để viết phần tổng quan/literature review.                                                                                                                                      | PDF trực tiếp: ([Miltos Allamanis][12])                                                      |
| Tham khảo thêm | **Graph-based, Self-Supervised Program Repair from Diagnostic Feedback** — Yasunaga, Liang, ICML 2020                                        | Phù hợp với cụm “context-aware bug detection/repair using graphs”, dù không tập trung riêng vào variable misuse. Có GNN + self-supervised program repair.                                                                                    | arXiv: ([arXiv][13]) / PDF trực tiếp: ([Stanford NLP][14])                                   |
| Tham khảo thêm | **Are Neural Bug Detectors Comparable to Software Developers on Variable Misuse Bugs?** — Richter et al., ASE 2022                           | Rất hợp phần đánh giá/so sánh vì kiểm tra neural bug detectors trên **variable misuse bugs** thực tế.                                                                                                                                        | ACM page: ([ACM Digital Library][15]) / PDF ACM: ([ACM Digital Library][16])                 |
